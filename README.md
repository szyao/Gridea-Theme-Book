# Theme Download

👍 Simply download this project to your local Gridea theme folder (usually .\Documents\Gridea\themes\).
👍只需将此项目下载到本地的Gridea主题文件夹（\Documents\Gridea\themes）。

- 这是一个有趣的主题，需要1200*900px才能获得完整体验
- 自适应主题，也适配手机
- 无评论系统
- 纯文字主题，无分类，无标签
- 纯纯适合写个日记啥的
